このソフトウェアには、（株）ＣＲＩ・ミドルウェアの「CRIWARE (R)」が使用されています。

DirectX11を使用しています
インストールされていない場合は下のURLから「DirectX End-User Runtime」をダウンロードをよろしくお願いします。
URL：https://www.microsoft.com/ja-jp/download/details.aspx?id=35

Visual Studio等でビルドをする場合は実行環境にあるasset/audio/shaderフォルダをソリューションファイルがある場所にコピー＆ペーストしてください。

cereal - A C++11 library for serializationを使用しています。
URL：https://uscilab.github.io/cereal/
